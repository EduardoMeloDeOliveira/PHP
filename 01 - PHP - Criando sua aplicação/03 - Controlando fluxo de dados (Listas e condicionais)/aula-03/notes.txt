Nessa aula, aprendemos sobre a variável argc, que nos informa a quantidade de parâmetros que passamos na linha de comando. Também vimos como usar o loop for para iterar sobre esses parâmetros e realizar uma operação, como somar as notas de um filme.

O for é uma estrutura de repetição que nos permite executar um bloco de código várias vezes, com um contador que é incrementado a cada iteração.

Para usar o for, precisamos definir três partes:

Inicialização: Define o valor inicial do contador.
Condição: Define a condição que deve ser verdadeira para que o loop continue.
Incremento: Define a operação que será realizada no contador a cada iteração.
Com o for, podemos automatizar tarefas repetitivas, como calcular a média das notas de um filme, sem precisar escrever o mesmo código várias vezes.
